# ✈️ Flight Price Prediction (Machine Learning Project) 

## Objective
Develop a machine learning model to predict flight prices based on different features such as airline, duration, stops, and travel time.

## Problem Statement 📊
Flight prices vary significantly depending on multiple factors, making it difficult for customers and companies to estimate fair prices.

This project aims to build a predictive model that helps estimate flight costs and supports better decision-making.

## 📂 Dataset
The dataset includes information about:
- Airline
- Source and destination
- Duration of the flight
- Number of stops
- Departure and arrival times
- Price (target variable)

## ⚙️ Process

### 1. Data Cleaning
- Handling missing values
- Correcting data types
- Removing inconsistencies

### 2. Feature Engineering
- Creation of time ranges
- Transformation of categorical variables
- Handling outliers

### 3. Exploratory Data Analysis (EDA)
- Price distribution analysis
- Identification of patterns and outliers
- Relationship between features and price

### 4. Modeling
- Train-test split
- Model training (e.g., Linear Regression, Decision Trees, Random Forest)
- Model evaluation using MAE, RMSE, and R²

## 📈 Key Insights
- Flight duration and number of stops strongly influence price
- Certain airlines consistently show higher pricing
- Price distribution is highly skewed with significant outliers

## 🛠️ Tools
- Python (pandas, numpy, matplotlib, sklearn)

## 💡 Business Impact
This model can help:
- Estimate fair flight prices
- Support pricing strategies
- Improve customer decision-making
